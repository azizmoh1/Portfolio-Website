Featured CAD Case Study

3D Printed Fidget Toy

Course: ME 347 - Computer-Aided Design Design Environment: SolidWorks (ANSI Inch) Manufacturing Target: FDM (Fused Deposition Modeling) 3D Printing

This project involved designing a portable 3D printed fidget toy using a parametric CAD workflow in SolidWorks. The device was engineered as a compact mechanical mechanism while satisfying strict design-for-manufacturing constraints for FDM printing. Standardized gear components were generated using the SolidWorks Toolbox and integrated into a fully constrained assembly environment. The final design demonstrates how theoretical gear geometry, parametric constraints, and additive manufacturing considerations can be combined into a functional mechanical device.

Project Overview

Project Type

Mechanical CAD Design / Rapid Prototyping

Tools Used

SolidWorks SolidWorks Toolbox FDM 3D Printing

Focus Areas

Parametric Modeling Mechanical Assemblies Design for Manufacturing

Outcome

Fully functional pocket-sized fidget toy designed for additive manufacturing.

Hybrid CAD Modeling Strategy

The assembly was created using a hybrid workflow combining Bottom-Up and Assembly-Driven design approaches. Individual components such as the sun gear, planet gears, and housing were first generated as discrete parts using the SolidWorks Toolbox. These components were then integrated into a parametric assembly environment where constraints and interference resolution were applied.

Gear Generation

Involute gears were generated with SolidWorks Toolbox rather than manually sketching tooth profiles. This ensured mathematically accurate involute geometry and consistent engagement behavior.

Diametral Pitch (DP): 24

Pressure Angle: 20deg

Tooth Count: Sun 14 | Planet 14 | Ring 42

Parametric Constraints and Gear Kinematics

The gear assembly depends on precise center distances and tangency between pitch circles. Center distance was enforced parametrically to preserve mesh quality and rotational consistency.

Center Distance Formula: C = (Ns + Np) / (2 x DP)

Using Ns = 14, Np = 14, DP = 24

Center Distance = 0.5833 inches

This distance constrained relative sun/planet axis placement in the assembly.

  • Concentric mates for shaft alignment
  • Coincident mates for planar positioning
  • Mechanical Gear Mates to enforce the 4:1 velocity ratio in SolidWorks motion

Design for Additive Manufacturing

Ideal CAD geometry was adjusted to account for real-world FDM process behavior and tolerance stacking.

Lateral Backlash Compensation

The Move Face command was used to offset tooth flanks by -0.005 inches. This reduced tooth thickness intentionally, creating physical backlash and preventing binding caused by over-extrusion.

Clearance Fits

Rotating shafts were given 0.015 inches diametrical clearance to produce running fits rather than interference fits. This enabled smooth post-print motion without extensive post-processing.

CAD Verification and Assembly Analysis

SolidWorks evaluation tools were used to validate assembly integrity before fabrication.

  • Interference Detection: identified and removed overlapping geometry between gear teeth.
  • Section View Analysis: verified axial spacing and internal clearances.

These checks reduced risk and improved confidence in first-pass print functionality.

Results

The completed CAD assembly demonstrates a compact mechanical fidget toy balancing theoretical gear geometry with practical additive manufacturing tolerances. The final model combines standardized Toolbox components with custom parametric adjustments to achieve smooth rotational motion in a 3D printed mechanism.

Lessons Learned

  • Parametric constraints are critical to robust mechanical assemblies.
  • Theoretical gear geometry must be adapted for manufacturable, real-world behavior.
  • FDM-specific tolerance compensation should be designed into the CAD model early.
  • SolidWorks evaluation tools are essential for pre-fabrication validation.
