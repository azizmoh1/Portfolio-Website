Active Thermal Management Simulation

In Progress | Thermodynamics + Controls

Project Overview

Building a simulation-driven thermal control model for EV battery or motor systems to maintain safe operating temperatures under variable duty cycles.

Problem Statement

Thermal loads vary dynamically with drive behavior; the system must detect and respond quickly to avoid overheating while minimizing unnecessary cooling power.

Design Process

  • Model heat generation from drive cycle input.
  • Implement fan/pump control logic based on temperature thresholds and hysteresis.
  • Evaluate system response for stability and control efficiency.

Lessons Learned
