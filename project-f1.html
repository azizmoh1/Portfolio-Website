Predictive Analytics for Formula 1 Race Dynamics

In Progress | Data Engineering & Feature Selection

Project Overview

Developing a machine-learning pipeline to forecast race outcomes by combining historical data with telemetry-driven performance indicators.

Problem Statement

Race outcomes depend on interacting variables (pace, tire wear, reliability, strategy); the challenge is extracting robust predictive signals from noisy, multi-source data.

Design Process

  • Extracting and cleaning telemetry with FastF1 API across multiple seasons.
  • Engineering features for degradation curves, pit strategy, and circuit behavior.
  • Planning predictive baselines with Random Forest / Gradient Boosting regressors.
